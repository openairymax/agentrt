## 如何成为贡献者

我们欢迎各种形式的贡献！

### 贡献方式

| 方式 | 说明 | 认可级别 |
|------|------|----------|
| **代码贡献** | 提交 PR 并被合并 | 核心贡献者 |
| **文档贡献** | 改进文档、翻译 | 文档贡献者 |
| **问题报告** | 报告 Bug、提建议 | 问题报告者 |
| **代码审查** | 参与 PR 审查 | 核心贡献者 |
| **社区支持** | 回答问题、帮助他人 | 社区贡献者 |

### 贡献流程

```
1. Fork 项目
2. 创建分支 (feature/xxx 或 bugfix/xxx)
3. 开发并测试
4. 提交 PR
5. 通过代码审查
6. 合并到主分支
7. 加入贡献者名单 🎉
```

详细流程请参阅 [CONTRIBUTING.md](./CONTRIBUTING.md)。

### 贡献者权益

| 权益 | 核心贡献者 | 活跃贡献者 | 普通贡献者 |
