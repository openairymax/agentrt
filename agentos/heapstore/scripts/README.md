# Heapstore Scripts

AgentOS Heapstore 模块的实用工具脚本。

## 脚本清单

| 脚本 | 说明 |
|------|------|
| `add_since_tags.py` | 为代码添加 @since 版本标签 |
| `performance_regression_detector.py` | 性能回归检测工具 |

## 用法

```bash
python scripts/add_since_tags.py
python scripts/performance_regression_detector.py
```
