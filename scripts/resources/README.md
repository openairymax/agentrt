# 资源与教程

`scripts/resources/`

## 概述

`resources/` 目录存放 AgentOS 项目的静态资源和交互式教程素材。

## 目录结构

```
resources/
├── README.md
├── images/                            # 静态图片资源
│   ├── AgentOS-desktop-preview.gif    # 桌面端预览动图
│   └── feishu-community-qr.png        # 飞书社区二维码
└── tutorial/                          # 教程引擎
    ├── tutorial_engine.py             # 交互式教程引擎
    └── new-contributor.json           # 新贡献者教程数据
```
